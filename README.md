# Countdown

A simple Android app to count down days to your upcoming events (exams, birthdays, deadlines, etc.).

## Features

- **Add events** with a name and a target date using a date picker
- **Countdown display** shows days remaining for each event on the main screen
- **Home screen widget** shows your nearest upcoming event's day countdown — just glance at your home screen!
- **Delete events** when they are no longer needed
- Simple, clean Material Design UI

## How to Use

1. Open the app and tap the **+** button to add a new event
2. Enter the event name (e.g. "Math Exam") and pick the date
3. The main screen shows all your events with days remaining (28, 27, 26…)
4. Add the **Countdown Widget** to your home screen for a quick glance

## Building

Open the project in Android Studio (Arctic Fox or later) and build the `app` module.

- Min SDK: 21 (Android 5.0)
- Target SDK: 34 (Android 14)
